# 中文标签配置说明

## 功能说明
本配置文件用于将 YOLO 模型检测结果中的英文标签自动转换为中文标签。

## 配置文件位置
`GUI/config/chinese_labels.json`

## 使用方法

### 1. 查看当前映射
打开 `chinese_labels.json` 文件，查看当前的英文到中文映射关系。

### 2. 添加新的映射
如果检测结果中出现了未映射的英文标签，可以在 `mapping` 部分添加新的映射：

```json
{
  "mapping": {
    "英文标签": "中文标签",
    "new_label": "新标签"
  }
}
```

### 3. 修改现有映射
直接编辑对应的中文翻译即可。

## 当前支持的坐姿类别

| 英文标签 | 中文标签 | 说明 |
|---------|---------|------|
| correct / normal / sitting_correct | 正确坐姿 | 标准坐姿 |
| forward / lean_forward / sitting_forward | 前倾 | 身体前倾 |
| backward / lean_back / sitting_backward | 后仰 | 身体后仰 |
| left / lean_left / sitting_left | 左倾 | 身体左倾 |
| right / lean_right / sitting_right | 右倾 | 身体右倾 |
| lying / lie_down / sitting_lying | 趴桌 | 趴在桌子上 |

## 注意事项

1. 修改配置文件后，需要**重启 GUI 程序**才能生效
2. 配置文件使用 UTF-8 编码，请确保保存时使用正确的编码
3. JSON 格式要求严格，注意逗号和引号的使用
4. 如果配置文件加载失败，程序会自动使用内置的默认映射

## 故障排除

### 问题：修改后没有生效
**解决方案**：
1. 检查 JSON 格式是否正确
2. 确认文件编码为 UTF-8
3. 重启 GUI 程序

### 问题：某些标签仍然显示英文
**解决方案**：
1. 查看检测结果中的英文标签名称
2. 在配置文件中添加对应的映射
3. 重启程序

## 示例

假设模型检测到的英文标签是 `slouching`（驼背），你想将其显示为"驼背"：

1. 打开 `chinese_labels.json`
2. 在 `mapping` 中添加：
```json
"slouching": "驼背"
```
3. 保存文件
4. 重启 GUI 程序

现在检测结果中的 `slouching` 就会显示为"驼背"了。
